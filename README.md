# Крестики-нолики (TicTacToe)

Android-игра в двух режимах.

## Возможности
- Игрок против игрока
- Игрок против компьютера (сложный уровень)
- Определение победителя
- Золотая победная линия
- Счет побед игроков
- Сброс счета игры

## Технологии
- Java
- Android SDK
